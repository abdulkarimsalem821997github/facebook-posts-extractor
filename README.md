# Facebook Posts Extractor

أداة متقدمة لاستخراج منشورات فيسبوك من الـ News Feed. تدعم اللغة العربية وتقدم بيانات دقيقة مع إحصائيات مفصلة.

## المميزات:
- ✅ استخراج المنشورات من الـ News Feed
- ✅ دعم اللغة العربية بالكامل
- ✅ إحصائيات مفصلة عن التفاعلات
- ✅ تصدير البيانات بصيغة CSV
- ✅ واجهة مستخدم سهلة
- ✅ كود Node.js منفصل للاستخدام المتقدم

## الاستخدامات:
- تحليل محتوى وسائل التواصل
- أبحاث السوق
- جمع البيانات للأكاديميين
- مراقبة العلامات التجارية
